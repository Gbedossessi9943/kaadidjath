# 🚀 Déploiement Rapide Kadidjath sur GitHub

## 📋 Étapes Simples (5 minutes)

### 1️⃣ Installer Git
- Téléchargez Git : https://git-scm.com/download/win
- Installez-le avec les options par défaut

### 2️⃣ Créer un Repository GitHub
1. Allez sur https://github.com
2. Cliquez sur **"New repository"**
3. Nom : `kadidjath-platform`
4. Description : `Plateforme e-commerce pour produits frais du Bénin`
5. **Public** (coché)
6. **NE PAS** cocher "Add README"
7. Cliquez sur **"Create repository"**

### 3️⃣ Déployer Automatiquement
1. Double-cliquez sur le fichier `deploy.bat`
2. Suivez les instructions
3. Entrez votre nom et email GitHub
4. Entrez l'URL de votre repository GitHub

### 4️⃣ Activer GitHub Pages
1. Sur votre repository GitHub, cliquez sur **"Settings"**
2. Dans le menu gauche, cliquez sur **"Pages"**
3. Source : **"Deploy from a branch"**
4. Branch : **"main"**
5. Folder : **"/ (root)"**
6. Cliquez sur **"Save"**

### 5️⃣ Attendre le Déploiement
- Patientez 2-5 minutes
- Votre site sera disponible à : `https://VOTRE_USERNAME.github.io/kadidjath-platform/`
